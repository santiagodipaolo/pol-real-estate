## Lo que se construyó esta sesión

### Scraper de Zonaprop (`app/scrapers/`)
- Playwright + extracción DOM con `data-qa` selectors
- ~25-30 listings/página, dedup por source+external_id
- CLI: `python -m scripts.scrape --pages 10 --operation sale`
- Celery task cada 6h (sale + rent)
- Pipeline: match barrio por nombre, conversión USD/ARS vía DolarAPI

### Motor de valuación (`app/valuation/`)
- XGBoost + quantile regression (intervalos de confianza)
- 13 features: superficie, ambientes, barrio_median, garages, expensas, etc
- 2 modelos: `valuation_sale_v1.joblib` + `valuation_rent_v1.joblib`
- CLI: `python -m scripts.train_model --operation sale`

### API endpoint
- `POST /api/v1/valuation/estimate` — valuación instantánea
- `GET /api/v1/valuation/metrics` — métricas del modelo
- `POST /api/v1/valuation/reload` — recargar tras reentrenamiento

### Frontend
- Página `/valuacion` con formulario + resultado visual
- Sidebar actualizado con link a Valuación

- Paginación: `-pagina-{n}.html`
